# 🏥 Patient Health Risk Analysis using NumPy

## 📌 Introduction
This project focuses on analyzing patient health data using Python and NumPy. It calculates a **health risk score** based on key medical parameters such as blood pressure, cholesterol, and sugar levels. The system then classifies patients into different risk categories and performs basic clustering and statistical analysis.

---

## 🚀 Features
1. **Data Loading**
   - Reads patient data from a CSV file using NumPy.

2. **Health Risk Score Calculation**
   - Computes risk score using:
     ```
     Risk Score = BP + Cholesterol + Sugar
     ```

3. **Risk Classification**
   - Categorizes patients into:
     - Low Risk
     - Medium Risk
     - High Risk

4. **Clustering**
   - Divides patients into two groups based on average BP:
     - Lower BP cluster
     - Higher BP cluster

5. **Statistical Analysis**
   - Calculates:
     - Average Age
     - Average BP
     - Maximum Cholesterol
     - Minimum Sugar Level
